# Homelab Nextcloud Server

Servidor personal autoalojado con Nextcloud en una máquina Ubuntu, configurado para almacenamiento, sincronización de archivos y acceso local.

## Objetivo
Este proyecto documenta la implementación y gestión de un servidor personal para:

- Almacenamiento de archivos
- Sincronización entre dispositivos
- Aprendizaje práctico de servidores reales

## Entorno
- Sistema operativo: Ubuntu
- Servicio: Nextcloud
- Red: Local (LAN)

## Implementación
- Instalación manual de Nextcloud
- Configuración de usuarios y almacenamiento
- Gestión del entorno en Linux
- Configuración básica de red (puertos y acceso)

## Configuración de red
- Acceso local dentro de la red
- Configuración de puertos (NAT / router)

## Retos
- Configuración de acceso remoto (Aún no resuelto)
- Manejo de puertos
- Instalación inicial y dependencias

## Aprendizajes
- Despliegue de servicios reales
- Fundamentos de red (puertos, NAT)
- Administración básica de servidores Linux
- Importancia de la seguridad en servicios expuestos

## Estado
Servidor en funcionamiento y en mejora continua.

## Nota de seguridad
Este repositorio no incluye información sensible como direcciones IP, credenciales o configuraciones privadas.
